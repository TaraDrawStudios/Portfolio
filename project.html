My Projects

MeowBot

MeowBot is a Discord bot that I created using Python and the discord.py library. It features a variety of commands, including fun cat facts, image generation, and moderation tools. The bot is designed to be user-friendly and customizable, allowing server administrators to tailor its functionality to their needs.

Birthday Cake

Birthday Cake is a simple web application that I created using HTML, CSS, and JavaScript. This was made for my father on his birthday where he was able to blow out the candles via microphone. The application also includes a fun animation of a birthday cake with candles that light up on the user's birthday.

Breast Cancer Prediction

Breast Cancer Prediction is a machine learning project that I created using Python. The project uses a dataset of breast cancer cases to train a model that can predict whether a tumor is malignant or benign. The model is evaluated using metrics such as accuracy, precision and can be used to assist in the diagnosis of breast cancer.

Miami Dade Teach Sign In

Miami Dade Teach Sign In is an application that I created using java. The application allows teachers in the Miami Dade school district to sign in and out of their classrooms using a simple interface. The application also includes a feature that allows administrators to view attendance records and generate reports.
