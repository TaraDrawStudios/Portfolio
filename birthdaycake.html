Birthday Cake

An Interactive Cake

The Birthday Cake is a fun and interactive project designed to celebrate birthdays in a unique way. It features a cute cake with a candle that can be blown out simply by blowing into the microphone. The project is built using HTML, CSS, and JavaScript, and it utilizes the Web Audio API to capture the user's breath and trigger the candle blowing animation.
