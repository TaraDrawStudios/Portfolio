MeowBot

A Discord Bot for Artists

MeowBot is a fun and interactive Discord bot designed to help me manage my art server. It provides various features such as welcoming new members, moderating content, and creating games and events. I built MeowBot using Python and the Discord.py library, allowing me to create custom commands and automate server management tasks. The bot has been a great way for me to engage with my community and share my love for art while also honing my programming skills.
